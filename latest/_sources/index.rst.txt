yagdal Documentation
====================

Yet another GDAL Python interface to  `GDAL <https://gdal.org/>`_ (translator library for raster and vector geospatial data formats).

GitHub Repository: https://github.com/yagdal/yagdal

.. note:: Minimum supported GDAL version is 3.4

.. note:: Minimum supported Python version is 3.8
